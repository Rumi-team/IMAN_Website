# IMAN Website

Modern bilingual (English/Farsi) website for the **Iranian-American Muslim Association of North America** (IMAN), serving the Los Angeles community.

## Design

The design system is documented in [DESIGN.md](./DESIGN.md). Open `design-preview.html` in a browser to see rendered fonts, colors, and realistic mockups.

**Design thesis:** "Isfahan meets Los Angeles" - Persian architectural colors (turquoise, lapis, gold) on warm limestone backgrounds. Bilingual English/Farsi as a first-class design feature, not a translation toggle.

## Phases

- **Phase 1:** Aesthetic website (current)
- **Phase 2:** Admin CMS for content management
- **Phase 3:** AI features

## Tech Stack

- Next.js (App Router)
- Vercel (deployment)
- Tailwind CSS

## Development

```bash
pnpm install
pnpm dev
```
